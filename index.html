Projetos

    To Do List

    A atividade modelo Aplicada pelo professor de Front End consistia em uma aplicação do tipo TODO list, que simula uma lista de tarefas a serem realizadas, conforme uma tarefa é realizada um estilo é aplicado para riscar a palavra. O objetivo consistia em construir essa mesma estrutura e adicionar as seguintes funcionalidades:

    1) Editar um texto (tarefa) já criado e atualizar isso na lista de tarefas do usuário.
    2) Criar uma opção (botão ou outra estrutura) que permita o usuário ocultar uma tarefa da lista, mas essa ação só pode ser feita com tarefas na concluídas.
    3) Criar uma estrutura de estilização (sem biblioteca adicionais) para deixar a aplicação mais bonita e criativa.
    4) Criar um repositório no GitHub e postar os arquivos no repositório e responder esse fórum com o link do seu repositório implementado.
